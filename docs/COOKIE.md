# Cookie setup

COOKIE нужен для доступа к app.send.tg.

## Как получить COOKIE

1. Открыть Chrome
2. Зайти на https://app.send.tg
3. Авторизоваться
4. Нажать F12
5. Открыть Application -> Cookies
6. Скопировать cookie строку

Можно использовать расширение Cookie-Editor.

COOKIE вставить сюда:

go-worker/.env

Пример:

COOKIE=...полная cookie строка...

Важно: COOKIE нельзя передавать другим людям.
